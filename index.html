Agenda (Vietnamese speaking)

08:30 AM - 08:35 AM
Opening
08:35 AM - 09:25 AM

.NET 6.0 - An Overview

Thien Nguyen

Highlight new features of .NET 6 and its ecosystem: C# 10, Minimal APIs, Hot Reload, ...

09:25 AM - 10:15 AM

Streamline .NET Microservices with Minimal APIs and C# 10

Thang Chung

What are the cool features of .NET 6 and C# 10 help to build modern applications? Subsequently, the presentation will also show you an example of how to build the greenfield Microservices Application from scratch by leveraging what .NET 6 ships with.

10:15 AM - 11:05 AM

Blazor WebAssembly - Perfect for SPA and much more

Thanh Le

- WebAssembly is a client-side technology that enables near-native performance without plug-ins in all modern browsers, including mobile.
- Blazor is a client-side library that uses .NET on WebAssembly to support Single Page Applications written in C# using Razor templates.
- .NET 6 is coming and, with it, some notable Blazor improvements.

11:05 AM - 11:55 AM

ML.NET, bring ML models from your terminal to Azure

Phi Huynh

Using ML.NET - a free, cross-platform and open source ML framework - to build ML models from experimentations in your Visual Studio Code & deploy to Microsoft Azure.

11:55 AM - 12:00 PM
Closing

Speakers

Thien Nguyen

Thien is Engineering Manager at NashTech, a Microsoft MVP and the creator of SimplCommerce. He is a passionate developer, a mentor and a speaker. His current interests are .NET Core, Azure and Microservices

Thanh Le

Thanh is Technical Architect at NashTech. He is also speaker, blogger, and mentor. Thanh's Medium page has more than 1000 followers with a thousand of views each month. His expertise is designing Microservice Architecture using Domain Driven Design.

Phi Huynh

Phi Huynh is currently working as R&D Manager at NashTech, building new skills for the company, help NashTech’s community up-to-date with latest technology trends (such as Blockchain, Microservices, DevOps, Cloud & AI). A technology enthusiast has been always looking for new fun things to try and loving to update latest technologies.

Thang Chung

Thang is a NashTech Senior Solution Architect and a Microsoft MVP. He passions to work on Open Source projects and mainly focuses on designing, implementing and deploying Microservices Applications as well as toolings and (F)OSS in the CNCF ecosystem.
